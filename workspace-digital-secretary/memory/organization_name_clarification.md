# 组织名称澄清

## 策略中心 = 社交信息安全中心
- **历史名称**：策略中心
- **当前正式名称**：社交信息安全中心
- **使用说明**：在文档和讨论中可能出现两个名称混用的情况，实际指向同一组织实体
- **上下文识别**：当看到"策略中心"时，应理解为当前的"社交信息安全中心"

## 相关术语关联
- **中场**：指代社交信息安全中心在前中后场协作体系中的定位
- **策略团队**：社交信息安全中心内部的策略职能团队
- **策略主线工作**：社交信息安全中心的核心业务方向

## 应用场景
- 文档归档和检索时需同时考虑两个名称
- 在知识库索引中建立名称映射关系
- 避免因名称差异导致的信息割裂
