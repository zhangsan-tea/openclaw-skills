# 微信GPA安全团队关于龙虾使用的反馈思考

**日期**: 2026-03-19  
**来源**: 与微信GPA安全同学的交流  

## 核心观点

安全团队认为目前龙虾的使用并不紧迫，主要原因如下：

1. **单环节能力优先**: 团队当前最需要提升的是单环节的能力，特别是**识别能力**——即从海量信息中准确找出目标信息
2. **识别能力的双重挑战**: 
   - 避免过多漏过（recall）
   - 避免过多误杀（precision）
3. **龙虾的定位理解**: 龙虾的作用是把各个环节串联起来，自动推动需要人工推动的事项
4. **当前阶段的局限性**: 在单环节能力尚未充分提升的情况下，串联各环节的意义还不够大
5. **价值空间不明确**: 目前还不确切知道龙虾在安全团队中的具体发挥作用的空间

## 备注
- 这不是最终定论，而是安全团队当前的一个思考角度
- 可作为安全团队AI使用策略的参考之一
