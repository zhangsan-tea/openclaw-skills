# 自建 Skills 目录

> **定义**：本目录汇总的是**用户主动发起建设**的业务 Skill（如周报编辑、迎检月报编辑），而非 WorkBuddy 系统预置的基础设施 Skill（如 computer-use、preview 等连接器类）。

## 区分标准

| 类型 | 来源 | 举例 | 归属 |
|------|------|------|------|
| **自建 Skill** | 用户主动发起建设 | 周报编辑、迎检月报编辑 | ✅ 收录于此 |
| **系统预置 Skill** | WorkBuddy 沙箱自带 | computer-use、preview、各种 connector | ❌ 不收录 |
| **官方市场 Skill** | 从市场安装 | 企业微信套件、腾讯会议 | ❌ 不收录 |

## 维护规则

1. **新增自建 Skill** → 在此目录下创建同名子目录 + 文档，更新本 README 清单
2. **更新 Skill** → 同步更新对应子目录文档，标注更新时间和变更内容
3. **废弃 Skill** → 标注 `status: 已废弃`，不移除文档（保留历史）
4. **GitHub 同步** → 每次变更后推送到 `zhangsan-tea/openclaw-skills` 仓库的 `workbuddy-skills/` 目录
