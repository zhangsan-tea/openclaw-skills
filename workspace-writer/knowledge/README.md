# 写作助手知识库（Knowledge Base）

本目录用于沉淀与你（lee）及写作助手相关的通用知识与可复用资产：风格规范、工作流程、常用模板与术语表。

- 适用范围：写作任务的“长期不变/相对稳定”的规则与素材
- 不放内容：一次性对话草稿、临时便签（请放在 memory/ 或项目文件）

## 结构
- index.md — 知识库总览与导航
- style-guide.md — 写作风格与表达规范
- workflows.md — 标准流程与确认清单
- glossary.md — 术语表/名词约定
- templates/ — 常用写作模板（文章、报告等）

## 变更约定
- 小改动：直接编辑相应文件并注明日期
- 大改动：在文件开头添加“变更记录”分节，说明背景/影响范围

## 配套
- memory/ 与 MEMORY.md：存放个性化、时效性的“记忆条目”，便于检索与复用
