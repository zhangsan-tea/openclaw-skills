# 《七夕》

2006年的七月  
我买来七只鸟儿  
想在夜里  
放他们飞越天空的河流

大暑之后是伏天  
漫长而闷热  
几千年了  
年年如此

这个夜晚热烈而凄凉  
不是因为别离  
是因为相见

在我们头顶的天空之河  
在群鸟之上  
他们聚了又散散了又聚  
几千年了，年年如此

讲故事的人先后长眠  
满街的广告牌  
无声喧哗  
不再有男耕女织  
我们的别离变的轻易  
相见，却更难

大暑之后是立秋  
金风吹拂  
背靠丰收的谷堆  
谁，还顾得上忧伤

七只鸟儿  
延着七个方向  
四散而去  
优美的弧线  
宛若古典的桥梁

（2006.7.31）
