# HEARTBEAT.md - 心跳检查规则

## 心跳检查规则（2026-03-15 更新）

### 核心原则

**正常情况**：
- 无任务 + 系统正常 = **保持沉默**，不发送任何信息

**需要报告**：
- 有进行中的任务 → 每小时报送进度
- 发现异常或错误 → 立即报告

### 实施规则

```
接收心跳信号
  ↓
检查：有任务吗？ / 有异常吗？
  ├─ 有 → 发送进度报告（每小时）
  └─ 否 → 保持沉默，不发送任何信息
```

### 禁止事项

- ❌ 禁止在正常情况下打扰用户
- ❌ 禁止没有实质内容的空回复
- ❌ 禁止回复"保持沉默"等无意义内容
