## 音频文件处理规则

**核心原则**：收到音频文件时，必须第一时间调用 audio-understanding 或相关技能来判断其意图，绝不能主观臆测。

**具体流程**：
1. **接收音频**：当用户发送音频文件（如 .ogg, .mp3, .wav）时，立即识别。
2. **调用技能**：使用 `openai-whisper` 等本地语音转文本工具，或相关音频理解技能，对音频内容进行客观分析。
3. **确认意图**：基于转录或分析结果，与用户确认音频的真实意图和上下文，再进行后续处理。
4. **禁止行为**：严禁在未分析音频内容的情况下，仅凭上下文或个人猜测来解读音频含义。

此规则旨在确保对用户输入的尊重和处理的准确性。
