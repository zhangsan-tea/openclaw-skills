# 出行日程提醒指南

## 🚄 火车出行

### 时间规划
- **到站时间**：发车前至少 **30 分钟**
- **出发时间**：北京市内大部分地区到车站约 1 小时 → 提前 **1 小时**从本地出发

### 日程设置
| 项目 | 时间 | 说明 |
|------|------|------|
| 日程开始 | 发车前 30 分钟 | 到达车站的时间点 |
| 日程结束 | 发车时间 | 列车实际发车时间 |
| 第一次提醒 | 发车前 60 分钟 | **出发提醒**：该出门了 |
| 第二次提醒 | 发车前 90 分钟 | 准备提醒：收拾东西、检查身份证 |

### 日程内容模板
```
🚄 [车次] [出发站]→[到达站]

【车票信息】
• 车次：XXX
• 出发站：XXX
• 到达站：XXX
• 发车时间：X月X日 XX:XX
• 取票号：XXXXXXXXXX

【出行建议】
• 提前 30 分钟到站
• 带身份证取票/进站
```

## ✈️ 飞机出行

### 时间规划
- **到机场时间**：起飞前至少 **60 分钟**（国内）或 **90 分钟**（国际）
- **出发时间**：北京市内到机场约 1-1.5 小时 → 提前 **2 小时**从本地出发

### 日程设置
| 项目 | 时间 | 说明 |
|------|------|------|
| 日程开始 | 起飞前 60 分钟 | 到达机场的时间点 |
| 日程结束 | 起飞时间 | 航班实际起飞时间 |
| 第一次提醒 | 起飞前 90 分钟 | **出发提醒**：该出门了 |
| 第二次提醒 | 起飞前 150 分钟 | 准备提醒：收拾行李、检查证件 |

### 日程内容模板
```
✈️ [航班号] [出发地]→[目的地]

【航班信息】
• 航班号：XXX
• 出发机场：XXX
• 到达机场：XXX
• 起飞时间：X月X日 XX:XX
• 值机柜台：XXX（如有）

【出行建议】
• 提前 60 分钟到机场（国内）/ 90 分钟（国际）
• 带身份证/护照
• 液体不超过 100ml

【交通提示】
• 机场快轨/出租车预计 X 分钟
```

---

## 📝 设置原则

1. **两次提醒**：准备提醒 + 出发提醒
2. **提醒间隔**：间隔 30-60 分钟，避免太紧凑
3. **日程主体**：以「到站/到机场时间」作为日程开始，而非发车/起飞时间
4. **重要信息**：车次/航班号、取票号/值机柜台、证件提醒
5. **颜色区分**：火车用青色(7)，飞机用蓝色(9)
