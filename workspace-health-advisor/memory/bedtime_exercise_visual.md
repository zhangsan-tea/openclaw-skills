# 睡前轻度运动方案（带动作说明）

**制定日期**: 2026-03-22
**目的**: 延长睡眠时间 + 放松腰部 + 强化核心
**时间**: 睡前 1-2 小时，约 10 分钟

---

## 📋 动作清单

### 1️⃣ 婴儿式放松

**时长**: 1-2 分钟

```
     ┌─────────────────┐
     │   婴儿式        │
     │                 │
     │    (●)          │ ← 头部贴地
     │   /│\           │
     │  / │ \          │ ← 手臂前伸
     │    │            │
     │   / \           │ ← 臀部坐脚跟
     │  ◎   ◎          │
     └─────────────────┘
```

**动作要点**：
- 跪坐，臀部坐在脚跟上
- 上半身向前趴下
- 双臂向前伸直，额头贴地
- 全身放松，深呼吸

**作用**：拉伸腰部，缓解紧张

---

### 2️⃣ 臀桥

**次数**: 10-15 次

```
     ┌─────────────────┐
     │   臀桥          │
     │                 │
     │  起始位置：      │
     │  ───●───        │ ← 头
     │     │           │
     │    ╱╲          │ ← 膝盖弯曲
     │   ◎  ◎         │ ← 脚踩地
     │                 │
     │  抬起位置：      │
     │  ───●───        │
     │      ╲          │
     │       ╲───      │ ← 臀部抬起
     │      ◎  ◎       │
     └─────────────────┘
```

**动作要点**：
- 仰卧，膝盖弯曲约90度
- 双脚平踩地面，与臀同宽
- 臀部用力向上抬起，直到大腿和上身成直线
- 在最高点保持 3 秒
- 缓慢放下，臀部轻触地面后再次抬起

**作用**：强化臀肌，减轻腰部负担

---

### 3️⃣ 鸟狗式（小狗撒尿式）

**次数**: 每边 5-8 次

```
     ┌─────────────────┐
     │   鸟狗式        │
     │                 │
     │    起始：        │
     │      (●)        │
     │     /│\         │
     │   ──┼──         │
     │     │           │
     │    / \          │
     │   ◎   ◎         │
     │                 │
     │    伸展：        │
     │  ←──(●)         │
     │      │          │
     │     ─┼─         │
     │      │  ──→     │
     │     ◎           │
     └─────────────────┘
```

**动作要点**：
- 四点跪地：双手在肩正下方，膝盖在臀正下方
- 同时伸出左手和右腿
- 手臂向前平伸，腿向后平伸
- 保持平衡 3-5 秒
- 缓慢收回，换另一边

**作用**：温和强化核心，改善平衡

---

### 4️⃣ 死虫式

**次数**: 8-10 次

```
     ┌─────────────────┐
     │   死虫式        │
     │                 │
     │    起始：        │
     │    (●)          │
     │     │           │
     │   ─┼─           │ ← 双臂指向天花板
     │     │           │
     │    ╱╲          │ ← 双膝并拢
     │   ◎  ◎         │
     │                 │
     │    卷起：        │
     │    (●)          │
     │    ╱│╲          │
     │   ╱┼╲           │ ← 后背离地
     │  ◎  ◎           │
     └─────────────────┘
```

**动作要点**：
- 仰卧，双臂指向天花板
- 双膝并拢，向腹部卷曲
- 上半身同时向前卷起
- 后背和臀部离开床面，悬空几秒
- 躺下时脚和大背不碰床
- **注意**：酸胀的应该是腹肌，不是腰肌！

**作用**：强化腹肌，稳定核心

---

### 5️⃣ 提踵（踮脚）

**次数**: 20-30 次

```
     ┌─────────────────┐
     │   提踵          │
     │                 │
     │   正常站立：     │
     │     ○           │
     │     │           │
     │     │           │
     │     │           │
     │    ╱╲          │
     │   ◎  ◎         │ ← 全脚掌着地
     │                 │
     │   踮起脚尖：     │
     │     ○           │
     │     │           │
     │     │           │
     │     │           │
     │    ╱ ╲         │
     │   ◎  ◎         │ ← 脚跟抬起
     └─────────────────┘
```

**动作要点**：
- 站立，双脚与臀同宽
- 慢慢抬起脚跟，用脚尖站立
- 在最高点停顿 1 秒
- 缓慢落下，脚跟轻触地后再次抬起

**作用**：简单抗阻，促进血液循环

---

### 6️⃣ 深蹲（坐椅子式）

**次数**: 10-15 次

```
     ┌─────────────────┐
     │   深蹲          │
     │                 │
     │   站立：        │
     │     ○           │
     │     │           │
     │     │           │
     │    ╱╲          │
     │   ◎  ◎         │
     │                 │
     │   下蹲：        │
     │     ○           │
     │    ╱│╲          │
     │   ◎──◎          │ ← 膝盖微超脚尖
     │  ▔▔▔▔▔         │ ← 像坐椅子
     └─────────────────┘
```

**动作要点**：
- 站立，双脚与肩同宽
- 臀部向后坐，像坐在椅子上
- 膝盖微微超过脚尖即可，不用太深
- 大腿与地面平行或略高
- 保持背部挺直，不要弯腰
- 站起时臀部用力

**作用**：下肢抗阻，增加睡眠动力

---

**次数**: 各 10 次

```
     ┌─────────────────┐
     │   高抬腿        │
     │                 │
     │   左腿：        │
     │     ○           │
     │    ╱│           │
     │   ◎ │           │ ← 左膝抬起
     │      │          │
     │     ◎           │
     │                 │
     │   右腿：        │
     │     ○           │
     │     │╲          │
     │     │ ◎         │ ← 右膝抬起
     │     │           │
     │     ◎           │
     └─────────────────┘
```

**动作要点**：
- 原地站立
- 轮流抬起左右膝盖
- 膝盖尽量抬到腰部高度
- 手臂可以配合摆动
- 节奏不要太快

**作用**：全身激活，增加疲劳感

## 📅 练习顺序

| 顺序 | 动作 | 时长/次数 |
|-----|------|----------|
| 1 | 婴儿式放松 | 1-2 分钟 |
| 2 | 臀桥 | 10-15 次 |
| 3 | 鸟狗式 | 每边 5-8 次 |
| 4 | 死虫式 | 8-10 次 |
| 5 | 提踵 | 20-30 次 |
| 6 | 深蹲 | 10-15 次 |
| 7 | 高抬腿 | 各 10 次 |

**总时长**: 约 10 分钟

---

## ⚠️ 注意事项

1. **不要太剧烈** — 睡前运动要温和
2. **腰痛明显时** — 只做婴儿式放松
3. **循序渐进** — 次数可以从少开始
4. **坚持最重要** — 每天 10 分钟比偶尔 1 小时有用
