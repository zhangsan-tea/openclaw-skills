# 睡前运动语音引导脚本（优化版）

**总时长**：约 10 分钟
**语音语言**：中文

---

## 1️⃣ 婴儿式放松（60-90秒）

**引导语**：
```
现在开始睡前运动。

第一个动作：婴儿式放松。

请跪坐在垫子上，臀部坐在脚跟上。
上半身慢慢向前趴下，双臂向前伸直，额头轻轻贴地。

全身放松，深呼吸。

现在开始计时，保持这个姿势 60 秒。

...（停顿30秒）...

已经过了 30 秒，继续保持放松。

...（停顿30秒）...

很好，婴儿式完成。
```

---

## 2️⃣ 臀桥（10-15次）

**引导语**：
```
第二个动作：臀桥。

请仰卧，膝盖弯曲约 90 度，双脚平踩地面。

准备开始。

吸气，抬起臀部，直到大腿和上身成一条直线。
在最高点保持 3 秒。1、2、3。
呼气，缓慢放下。

再来一次。
抬起，保持。1、2、3。放下。

继续做，注意用臀部发力。
...（重复 10-15 次）...

臀桥完成，准备下一个动作。
```

---

## 3️⃣ 鸟狗式（每边 5-8 次）

**引导语**：
```
第三个动作：鸟狗式。

请四点跪地，双手在肩膀正下方，膝盖在臀部正下方。

先做左边。
伸出左手和右腿，保持平衡。
手臂向前伸，腿向后伸。
保持 5 秒。1、2、3、4、5。
收回。

换右边。
伸出右手和左腿，保持平衡。
保持 5 秒。1、2、3、4、5。
收回。

继续左右交替做。
...（重复引导）...

鸟狗式完成。
```

---

## 4️⃣ 死虫式（8-10 次）

**引导语**：
```
第四个动作：死虫式。

请仰卧，双臂指向天花板，双膝并拢。

吸气，双膝向腹部卷曲，上半身同时向前卷起。
后背和臀部离开地面，悬空保持 3 秒。
呼气，缓慢躺下，但脚不要碰到床。

再来一次。
卷起，保持。1、2、3。
放下。

继续做，注意腹肌发力，不是腰。
...（重复 8-10 次）...

死虫式完成。
```

---

## 5️⃣ 提踵（20-30 次）

**引导语**：
```
第五个动作：提踵，也就是踮脚。

请站立，双脚与臀同宽。

慢慢抬起脚跟，用脚尖站立。
在最高点停顿 1 秒。
缓慢落下。

继续做，节奏不要太快。
抬起、落下、抬起、落下。

...（重复 20-30 次）...

提踵完成。
```

---

## 6️⃣ 深蹲（10-15 次）

**引导语**：
```
第六个动作：深蹲。

请站立，双脚与肩同宽。

臀部向后坐，像坐在椅子上。
膝盖微微超过脚尖即可。
保持背部挺直。
在大腿与地面平行时，保持 2 秒。1、2。
站起，臀部用力。

再来一次。
下蹲，保持。1、2。
站起。

继续做，注意不要弯腰。
...（重复 10-15 次）...

深蹲完成。
```

---

## 7️⃣ 高抬腿（各 10 次）

**引导语**：
```
最后一个动作：高抬腿。

原地站立，准备开始。

左膝抬起，尽量到腰部高度。放下。
右膝抬起。放下。

左右交替，继续做。
左、右、左、右。

...（重复各 10 次）...

高抬腿完成。
```

---

## 结束语

**引导语**：
```
太棒了！今天的睡前运动全部完成。

你已经为良好的睡眠做好了准备。
现在可以放松休息了。

晚安，祝你一夜好眠！
```
