# 周报编辑 Skill

## 📋 简介

这是一个基于17条编辑规则的周报编辑技能，专门用于政务类、工作汇报类稿件的精简优化。该技能源自实际工作经验的总结，经过多个案例验证，能够有效提高周报的信息密度和可读性。

## 🚀 快速开始

### 1. 技能核心
- **17条编辑规则**：涵盖内容精简、句式优化、结构优化、语言表达
- **标准工作流程**：评估→编辑→优化→整理
- **实用模板**：提供标准输出格式

### 2. 使用方法
1. 阅读`SKILL_weekly_report_editing.md`了解17条规则
2. 参考`示例_团队周报_修订稿.md`学习应用方法
3. 按照"原文+修改稿+理由"格式进行编辑
4. 生成最终Word格式文档

### 3. 关键特性
- **保留原文顺序**：不改变原始结构，便于对照
- **逐段编辑**：每段独立处理，清晰展示修改过程
- **规则应用**：明确标注每条规则的应用
- **评估建议**：提供"拟报老板部分"优化建议

## 📝 17条编辑规则摘要

### 内容精简（4条）
1. 删除重复内容
2. 删除显而易见内容
3. 删除虚夸表述
4. 删除废话冗余

### 句式优化（4条）
5. 大幅精简复杂句式
6. 删除重复词语
7. 删除冗余连接词
8. 保留部门缩写（用户偏好）

### 结构优化（4条）
9. 避免标题和正文重复
10. 删除时间背景词
11. 简化并列句式
12. 删除重复主语

### 语言表达（5条）
13. 简化动词表达
14. 删除强调性副词
15. 删除不必要的指代词
16. 简化介词短语
17. 调整句子结构

## 🔧 工作流程

### 第一步：整体评估
- 阅读全文，了解整体内容和结构
- 识别重点事项和关键信息
- 评估"拟报老板部分"是否合理

### 第二步：逐段编辑
- 保留原文顺序，不调整段落
- 每段按"原文+修改稿+理由"格式呈现
- 逐句应用17条规则

### 第三步：重点优化
- 确保"拟报老板部分"包含最重要事项
- 高风险事项必须包含（罚款、通报、领导批示）
- 保留关键数据，删除冗余描述

### 第四步：格式整理
- 统一标题层级、编号、标点
- 检查部门缩写、专业术语一致性
- 通读检查流畅性

## 💡 使用技巧

### 新手建议
- 先从规则1-4开始练习（内容精简）
- 重点关注规则9（避免标题正文重复）
- 保留专业术语和关键数据
- 不要过度精简，保持核心信息完整

### 进阶技巧
- 根据稿件类型调整规则应用强度
- 政务稿件侧重规则3、8、9
- 工作汇报侧重规则2、4、14
- 风险通报侧重规则5、13、17
