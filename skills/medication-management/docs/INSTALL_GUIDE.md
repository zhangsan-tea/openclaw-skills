## 🚀 快速安装（3 步）

### 第 1 步：运行安装脚本

在终端输入：

```bash
cd /Users/lee/.openclaw/skills/medication-management
bash scripts/install.sh
```

**脚本会自动完成**：
- 创建药品清单文件
- 创建配置文件
- 设置定时任务
- 测试 Google Drive 连接

**预计时间**：2-3 分钟

---

### 第 2 步：授权 Google Drive

脚本会提示：

```
请打开以下链接授权 Google Drive 访问：
https://accounts.google.com/o/oauth2/auth?...
```

**操作**：
1. 复制链接，在浏览器打开
2. 登录 Google 账号
3. 点击"允许"授权
4. 授权成功后回到终端

---

### 第 3 步：录入药品

**方式 1：拍照录入**（推荐）

1. 打开药品包装
2. 拍摄保质期和药品名称
3. 发送到 Telegram
4. 系统自动识别并录入

**方式 2：手动录入**

```
用户：添加药品 - 布洛芬，保质期 2028-06-30
```

## ❓ 常见问题

### Q1: 授权失败怎么办？

**症状**：点击授权链接后显示错误

**解决**：
1. 检查网络是否正常
2. 确认 Google 账号密码正确
3. 重新运行安装脚本：`bash scripts/install.sh`

---

### Q2: 没有收到每周提醒？

**症状**：周日上午没有收到药品检查报告

**解决**：
1. 检查 Telegram 是否正常运行
2. 检查 OpenClaw 是否运行：`openclaw gateway status`
3. 手动触发一次：`cron run --job-id <药品检查 job ID>`

---

### Q3: 药品信息录入错误？

**症状**：保质期或药品名称错了

**解决**：
```
你：更新 med_001 的保质期为 2028-01-01
```

或者删除重新添加：
```
你：删除 med_001
你：添加药品 - XXX，保质期 YYYY-MM-DD
```

---

### Q4: 误删了药品怎么办？

**解决**：
1. 系统每天有自动备份
2. 运行恢复脚本：`bash scripts/restore_backup.sh`
3. 或手动从备份文件恢复

---

### Q5: 多个用户如何使用？

**说明**：当前版本支持单用户，多用户版本开发中

**临时方案**：
- 每个用户使用独立的 OpenClaw 实例
- 或共享一个账号（不推荐）
