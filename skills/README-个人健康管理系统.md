# 个人健康管理系统

> 基于 OpenClaw 构建的模块化健康管理方案

---

## 🎯 设计理念

这是一个**模块化**的个人健康管理系统，每个功能独立成单元，你可以按需选择安装。

**核心原则：**
- 我是教练，不是医生 — 不诊断、不开药方
- 关注长期习惯，而非短期效果
- 系统性健康视角 — 身体、心理、生活方式综合考虑
- 个性化适配 — 在你能力范围内提供最适合的方案

---

## 📦 模块选择

| 模块 | 功能 | 适合人群 | 难度 |
|------|------|----------|------|
| **健康顾问理念** | 健康管理的思维框架和方法论 | 所有人 | ⭐ |
| **药品管理** | 药品清单、过期检查、用药提醒 | 有家庭药箱的人 | ⭐⭐ |
| **健康数据自动化** | Apple Health 导出、周度报告 | iPhone + Apple Watch 用户 | ⭐⭐⭐ |
| **日程提醒** | 截图/文字 → 自动创建日历提醒 | 所有人 | ⭐ |

## 📥 安装方式

### 方式一：按需安装（推荐）

```bash
# 进入 OpenClaw skills 目录
cd ~/.openclaw/skills

# 只安装你需要的模块
git clone --depth 1 https://github.com/zhangsan-tea/openclaw-skills.git temp
cp -r temp/medication-management .          # 药品管理
cp -r temp/health-advisor-skill-extraction . # 健康顾问理念
cp -r temp/health-automation .               # 健康数据自动化
cp -r temp/calendar-reminder .               # 日程提醒
rm -rf temp
```

### 方式二：全部安装

```bash
cd ~/.openclaw/skills
git clone --depth 1 https://github.com/zhangsan-tea/openclaw-skills.git temp
cp -r temp/medication-management .
cp -r temp/health-advisor-skill-extraction .
cp -r temp/health-automation .
cp -r temp/calendar-reminder .
rm -rf temp
```

## 📋 模块详情

### 1️⃣ 健康顾问理念
**目录：** `health-advisor-skill-extraction`

**功能：**
- 健康管理的思维框架
- 工作原则和最佳实践
- 风险管理和边界意识

**适合：** 想了解健康管理方法论的人

---

### 2️⃣ 药品管理
**目录：** `medication-management`

**功能：**
- 家庭药品清单维护
- 过期药品自动检查
- 用药提醒

**触发词：** 药品、过期、用药提醒

**适合：** 有家庭药箱需要管理的人

---

### 3️⃣ 健康数据自动化
**目录：** `health-automation`

**功能：**
- Apple Health 数据自动导出
- 每周健康分析报告
- 睡前运动提醒
- 药品定期检查

**前置条件：**
- iPhone + Apple Watch
- Health Auto Export App
- Google 账号

**适合：** 有 iPhone + Apple Watch 的深度用户

---

### 4️⃣ 日程提醒
**目录：** `calendar-reminder`

**功能：**
- 截图/文字自动识别
- 智能提取关键信息
- 一键创建 Google 日历

**触发词：** 截图、预订、车票、机票、会议、提醒、日程

**适合：** 所有人

## ⚠️ 注意事项

- **医疗免责声明**：本系统提供的健康分析仅供参考，不能替代专业医疗建议
- **数据隐私**：健康数据存储在个人 Google Drive，仅授权用户访问
- **模块独立**：每个模块可独立运行，不强制安装全部
