# 12306火车票查询技能

## 描述
通过MCP Server 12306查询实时火车票信息，包括余票、车次、时刻、票价等详细信息。

## 触发条件
当用户需要查询火车票信息时使用此技能，特别是：
- 查询特定日期的火车票余票情况
- 查找符合时间要求的高铁/动车车次
- 获取准确的发车/到达时间和票价
- 需要实时票务信息而非静态时刻表

## 使用方法
通过HTTP API直接调用本地部署的MCP Server 12306服务。

### 主要功能
1. **query-tickets**: 查询余票/车次/座席/时刻一站式信息
2. **search-stations**: 智能车站搜索（支持中文名、拼音、三字码）
3. **query-ticket-price**: 查询票价详情
4. **query-transfer**: 中转换乘方案查询

### API端点
- 服务器地址: http://127.0.0.1:8001
- MCP端点: /mcp
- 健康检查: /health

## 参数说明
### query-tickets 参数
- `from_station`: 出发站（必填）
- `to_station`: 到达站（必填）  
- `train_date`: 出发日期，格式 YYYY-MM-DD（必填）

### 车站名称处理
- 支持城市名（如"北京"）→ 自动包含所有相关车站（北京/北京西/北京南等）
- 支持具体车站名（如"北京西站"）
- 支持三字码（如"BJP"）

## 返回数据结构
返回完整的车次信息，包括：
- 车次号（G/D/K/Z/T等）
- 出发/到达车站和时间
- 历时
- 各席别余票状态（有/无/数字）
- 票价信息

## 使用示例
```bash
# 查询北京到原平 2026-04-03 的车票
curl -X POST http://127.0.0.1:8001/mcp \
  -H "Content-Type: application/json" \
  -d '{
    "method": "query-tickets",
    "params": {
      "from_station": "北京",
      "to_station": "原平",
      "train_date": "2026-04-03"
    }
  }'
```

## 注意事项
- 所有时间均为北京时间（Asia/Shanghai）
- 余票信息为实时数据，可能随时变化
- 建议查询后立即进行购票操作
- 支持高铁(G)、动车(D)、直达(Z)、特快(T)、快速(K)等所有车次类型
