# 周报编辑 Skill

## 🎯 技能概述
基于17条编辑规则的周报编辑技能，专门用于政务类、工作汇报类稿件的精简优化。

## 📝 17条编辑规则速查

### 内容精简（4条）
1. **删除重复内容** - 同一概念多种表达，只保留核心
2. **删除显而易见内容** - 大家都知道，无需赘述
3. **删除虚夸表述** - 过于华丽，不实在
4. **删除废话冗余** - 不影响核心意思

### 句式优化（4条）
5. **大幅精简复杂句式** - 用简洁表达替换复杂句式
6. **删除重复词语** - 删除重复出现的同一概念
7. **删除冗余连接词** - 删除"也"、"该议题"等
8. **保留部门缩写** - 根据用户偏好保留（内部文稿）

### 结构优化（4条）
9. **避免标题和正文重复** - 标题已明确，正文不重复
10. **删除时间背景词** - 删除"近期"、"当前"等
11. **简化并列句式** - "除了...还从..."合并
12. **删除重复主语** - 同一对象短时间多次出现

### 语言表达（5条）
13. **简化动词表达** - 较长动词词组缩短
14. **删除强调性副词** - 删除"快速"、"积极"等
15. **删除不必要的指代词** - 删除"此事"、"此事件"等
16. **简化介词短语** - "为理由"简化为"为由"
17. **调整句子结构** - 重新组织句子成分

## 🔄 标准工作流程

### 第一步：整体评估
1. 阅读全文，了解整体内容和结构
2. 识别重点事项和关键信息
3. 评估"拟报老板部分"是否合理

### 第二步：逐段编辑
1. 保留原文顺序，不调整段落
2. 每段按"原文+修改稿+理由"格式呈现
3. 逐句应用17条规则

### 第三步：重点优化
1. 确保"拟报老板部分"包含最重要事项
2. 高风险事项必须包含（罚款、通报、领导批示）
3. 保留关键数据，删除冗余描述

### 第四步：格式整理
1. 统一标题层级、编号、标点
2. 检查部门缩写、专业术语一致性
3. 通读检查流畅性

## 💡 使用技巧

### 新手建议
1. 先从规则1-4开始练习（内容精简）
2. 重点关注规则9（避免标题正文重复）
3. 保留专业术语和关键数据
4. 不要过度精简，保持核心信息完整

### 进阶技巧
1. 根据稿件类型调整规则应用强度
2. 政务稿件侧重规则3、8、9
3. 工作汇报侧重规则2、4、14
4. 风险通报侧重规则5、13、17

### 质量检查
1. 通读修订稿，检查流畅性
2. 对照原文，确保核心信息不丢失
3. 检查专业术语是否正确保留
4. 确认"拟报老板部分"包含重要事项

> 传播局调研公司正能量宣传
> 央办传播局局长沈军带队到公司调研，团队介绍腾讯两会宣传工作，以及正能量内容的传播、网络名人的调动使用、2026年的正能量行动计划等；也探讨了如何更好地利用新技术提升正能量内容的传播效果，更有效地利用平台资源进行宣传。传播局高度认可腾讯长期以来在正能量宣传的支持与贡献，特别是强大的产品矩阵、策划能力和技术实力。下一步，团队将加强与传播局沟通，做好2026媒体论坛相关活动，持续放大正向价值。

### 修改建议稿
> 传播局调研公司正能量宣传
> 央办传播局局长沈军带队到公司调研，团队介绍了两会宣传工作、利用新技术提升正能量传播等工作。传播局高度认可腾讯长期以来在正能量宣传的支持与贡献。下一步，团队将加强与传播局沟通，做好2026媒体论坛等活动。

### 修改理由
1. **规则1：删除重复内容** - 删除"正能量内容的传播、网络名人的调动使用、2026年的正能量行动计划等"，只保留"利用新技术"
2. **规则2：删除显而易见内容** - 删除"更有效地利用平台资源进行宣传"
3. **规则3：删除虚夸表述** - 删除"特别是强大的产品矩阵、策划能力和技术实力"
4. **规则4：删除废话冗余** - 删除"，持续放大正向价值"
5. **规则5：大幅精简复杂句式** - 简化"也探讨了如何更好地利用新技术提升正能量内容的传播效果"为"利用新技术提升正能量传播"
6. **规则7：删除冗余连接词** - 删除"以及"、"等"
7. **规则9：避免标题和正文重复** - 标题已包含"传播局调研"，正文中不再重复
