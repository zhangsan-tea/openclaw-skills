# 药品管理系统 - 简化版用户手册

> 📖 只保留最常用的功能，5 分钟上手

---

## 🎯 这个系统能做什么？

1. **自动检查过期药品** - 每周日早上 9 点自动提醒
2. **管理药品清单** - 记录家里有什么药、什么时候过期
3. **拍照存档** - 给药品拍照，方便查找
4. **用药提醒** - 提醒按时吃药（可选）

---

## ❓ 常见问题

### Q: 没有收到每周提醒？

**解决**：
1. 检查系统状态：`bash scripts/check_status.sh`
2. 手动触发一次：`cron run --job-id <JOB_ID>`

---

### Q: 药品信息错了？

**解决**：
```
你：更新 med_001 的保质期为 2028-01-01
```

---

### Q: 误删了药品？

**解决**：
```bash
# 从备份恢复
cp medications/backups/medications_backup_最新日期.csv medications/medications.csv
```

---

### Q: 如何添加多个药品？

**方式 1：逐条添加**
```
添加药品 - 布洛芬，保质期 2028-06-30
添加药品 - 感冒清热颗粒，保质期 2026-01-31
添加药品 - 金银花口服液，保质期 2027-08-31
```

**方式 2：批量导入**
```bash
# 准备 CSV 文件，然后：
cat batch_import.csv >> medications/medications.csv
```

## 💡 小技巧

### 快速录入药品

**拍照 + 文字**：
1. 拍摄药品包装（特别是保质期）
2. 发送到 Telegram
3. 发送文字：`添加药品 - XXX，保质期 YYYY-MM-DD`

---

### 按位置整理

**分区建议**：
- **A 区**：常用药（感冒、发烧、止痛）
- **B 区**：肠胃药、保健品
- **C 区**：外用药、特殊药品

**录入时标注**：
```
添加药品 - 布洛芬，保质期 2028-06-30，存放位置 A 区
```

---

### 设置用药提醒

```
你：设置用药提醒 - 布洛芬，每天 3 次，饭后
```

系统会每天提醒你按时吃药。
