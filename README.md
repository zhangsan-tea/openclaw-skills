# 自建 Skills 知识库 (Obsidian Vault)

> 用 Obsidian 管理的**用户自建 Skill** 文档库。收录用户主动发起建设的业务 Skill。

## 目录结构

```
obsidian-vault/
├── README.md                    # 本文件
└── 自建Skills/
    ├── README.md                # 自建 Skill 总索引
    └── 同步规则.md               # 同步规则
```

## 什么是"自建 Skill"

用户主动发起建设的业务 Skill，区别于系统预置的基础设施 Skill 和官方市场安装的 Skill。

详见 `自建Skills/README.md`。

## GitHub 同步

- 仓库：[zhangsan-tea/openclaw-skills](https://github.com/zhangsan-tea/openclaw-skills)
- 路径：`workbuddy-skills/`

## 如何使用

1. 用 Obsidian 打开本目录作为 Vault
2. 创建自建 Skill 后，在 `自建Skills/` 下创建对应文档
3. 推送 GitHub：`git push origin main`
