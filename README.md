# my-ai-skills

Reusable Agent Skills (sanitized, no secrets).

## Included
- `voice-analysis/` — Voice analysis + humming/song recognition + emotion inference
- `medication-management/` — Household medication inventory + expiry checks + reminders
- `health-automation/` — Apple Health export automation + weekly reports + reminders

## Secrets
Do not commit secrets. Store API keys locally (e.g. `<OPENCLAW_WORKSPACE>/memory/api_keys.json`) or env vars.
