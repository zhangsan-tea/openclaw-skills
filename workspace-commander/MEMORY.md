# 养虾心得（精华）

## OpenClaw 产品定位
- **EasyClaw**：一键安装，稳定，但无法接入外部模型
- **OpenClaw**：灵活定制，多模型支持，工具生态丰富
- **Box**：内网连通 + 安全沙盒 + 浏览器控制，本质是运行在本机的 OpenClaw，非云端非纯内网，目前无法便捷配置多 Agent

## 使用原则
- 重要系统操作前，先判断**风险度**和**必要性**
- "养虾一定会挂掉"：接受故障，不挫败
- "把AI当员工"：明确指令、充分授权、结果导向
- 重要信息要明确要求存到 md 里

## 工具分工速查
- **一般问询**：Gemini/ChatGPT
- **内网信息**：Box（IWiki/KM/KNOT）
- **程序开发**：Codebuddy/Workbuddy
- **语音输入精确度**：豆包 > 微信 > SaySo > Typeless
